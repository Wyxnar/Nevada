## Info

A minimalist PowerShell tool for secure text encryption and decryption (Project II).

## Prerequisites
OS: Windows 11 (To verify, press Win + R, type "winver", and hit Enter).
PowerShell: Version 5.1 or higher (Pre-installed on Windows 11).

## Installation & Usage
1. Enter your desktop, Right Click, click New and create a Text Document.
2. Open the document and paste the code provided in the "Nevada.ps1" file.
3. Click File > Save As. In "Save as type", choose "All Files (*.*)". Name the file and ensure it ends with .ps1 (Example: Nevada.ps1).
4. Open Windows PowerShell in the folder where you saved the file.
5. Type the following command to execute the encryption protocol (replace FILENAME with your file's name):
```bash
Set-ExecutionPolicy Bypass -Scope Process; & ".\FILENAMEHERE.ps1"
